# go-parking

Go-parking — сервис для обработки изображений с парковок и интеграции с Telegram.

## Возможности
- Загрузка и обработка изображений парковки через веб-форму
- Автоматическое определение занятости парковочных мест по полигонам
- **🆕 Интерактивный редактор полигонов** - создание, редактирование и калибровка полигонов в браузере
- **🆕 Предварительный просмотр полигонов парковочных мест**
- **🆕 Улучшенная обработка изображений с адаптивными алгоритмами**
- **🆕 Многомасштабный анализ для повышения точности**
- **🆕 Адаптивные пороги в зависимости от условий освещения**
- Визуализация результата на изображении
- Отправка результата в Telegram (новое сообщение или обновление существующего)

- Автоматический анализ яркости, контраста и шума

- Файл настроек `advanced_config.json`
- API endpoint `/config` для просмотра настроек

## Быстрый старт

### Локальный запуск
```bash
# Установите зависимости
make build

# Запустите сервер
./cmd/go-parking/go-parking
```

Сервер будет доступен на http://localhost:9991

### Сборка и публикация контейнера

- Сборка локального образа:
  ```bash
  make build
  ```
- Публикация в Docker Hub:
  ```bash
  make publish
  ```

### Тесты и линт
```bash
make test
make lint
```

## Использование
- Откройте http://localhost:9991/form для загрузки изображения.
- **🆕 Откройте http://localhost:9991/preview для предварительного просмотра и редактирования полигонов.**
- Откройте http://localhost:9991/config для просмотра конфигурации.
- Заполните поля target (chat_id), token (bot token), отметьте day (если день), выберите файл и отправьте.
- Результат будет отправлен в Telegram.

## API для работы с полигонами
- `GET /api/polygons` - получить список всех полигонов
- `POST /api/polygons/save` - сохранить новые полигоны в конфигурацию

### Пример использования API:
```bash
# Получить текущие полигоны
curl http://localhost:9991/api/polygons

# Сохранить новые полигоны
curl -X POST http://localhost:9991/api/polygons/save \
  -H "Content-Type: application/json" \
  -d '{"polygons": [{"id": 0, "points": [{"x": 100, "y": 100}, {"x": 200, "y": 100}, {"x": 200, "y": 200}, {"x": 100, "y": 200}]}]}'
```

## � Новая функция: Интерактивный редактор полигонов

### Что это?
Мощный веб-редактор для создания, редактирования и калибровки полигонов парковочных мест прямо в браузере. Больше не нужно вручную редактировать координаты в коде!

### Основные возможности:
- **Визуальное редактирование**: Создавайте и изменяйте полигоны прямо на изображении
- **Режимы работы**: Выбор, добавление, перемещение, изменение размера, поворот
- **Автосохранение**: Автоматическая генерация файла `polygons.go` с backup'ами
- **Калибровка**: Точная настройка масштаба и смещения полигонов
- **REST API**: Программный доступ к управлению полигонами

### Как использовать:
1. Перейдите на http://localhost:9991/preview
2. Загрузите изображение парковки
3. Настройте масштаб и смещение если нужно
4. Используйте инструменты для редактирования полигонов
5. Сохраните изменения - система автоматически обновит файл `polygons.go`

**📖 Подробная инструкция:** [POLYGON_EDITOR.md](POLYGON_EDITOR.md)
